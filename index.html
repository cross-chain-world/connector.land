The Public Interledger.

This website aims to publish statistics about version 2 of the public Interleger network.

Version 1 of the network existed in 2016-2017, and used earlier versions of various protocols in the stack.

Version 2 consists of a number of servers which pass the following twelve tests:

  1. Run on a fully qualified domain name (domain or subdomain), reachable via DNS and IPv4 over the public internet.
  2. Announce a WebFinger record for the host, with:
    • property 'https://interledger.org/rel/publicKey': an ED25519 public key, e.g. 'FP5hUuESKM1ruif-fkelYNSTyt7qTd733sW3wBt73Ro'
    • property 'https://interledger.org/rel/protocolVersion': the string 'Compatible: ilp-kit v2.0.0-alpha'
    • link to the peers rpc uri, with rel 'https://interledger.org/rel/peersRpcUri'
  3. On the peers rpc uri, repond as follows:
      If
    • Repond to `send_transfer` requests, as specified in [ilp-plugin-virtual].
    • Fulfill SPSP payments for its own FQDN (with reasonable uptime and response time) as specified in [spsp]:
